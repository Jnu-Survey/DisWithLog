# DisWithLog

##  项目介绍：

暂时借助面向软件工程大作业编程的契机，做一个在线调查问卷管理系统。

相比较于市面上的收费管理软件，我们的目标是：在满足基础的功能下，基础版源码开源

方便各位大佬面向需求进行**二次开发**～ （您的一个个star✨是对我们的成功的认可）

- 前端开源地址：https://github.com/Jnu-Survey/WeChatFrontEnd
- 后端开源地址：https://github.com/Jnu-Survey/GinBackEnd

> 后期的打算是做商业版本，商业版因为涉及隐私🔏便不会进行开源。敬请谅解

##  项目定位

- 解决痛点：

1. 五六年前的网站问卷调查系统 <-------> 当今干啥都是限制收费的管理系统

2. 五六年前的鸡肋调查问卷系统 <-------> 当前很多功能都用不上的收费管理系统
3. ...

##  技术栈：

###  前端：

- 开发框架：uni-app
- 组件库：Vant-Weapp
- 语法扩展：Less
- 代码规范：ESLint、Prettier

###  后端：

- 主语言：Golang
- 开发框架：Gin
- 数据库：MongoDB Mysql
- 中间件：Redis
- 对象存储：七牛云COS

###  部署：

- WEB服务：Nginx
- Docker容器
- CDN加速
